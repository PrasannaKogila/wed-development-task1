# CSE Courses Webpage Project

This is a **CSE Courses webpage** project built as part of my training at **ApexPlanet Software Pvt. Ltd.**  

## Features
- HTML structured webpage with headings, paragraphs, and course sections
- CSS styling for layout, fonts, colors, and semi-transparent course boxes

- Creating interactive elements
